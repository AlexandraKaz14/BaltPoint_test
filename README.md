## Инструкция по запуску проекта локально

1. Копируем файл с переменными окружения
```
cp .env.example .env
```
2. Генерируем ключ для работы Laravel
```
php artisan key:generate 
```
3. Настроить подключение к БД в файле .env 
4. Запустить миграции и сидеры
```
php artisan migrate --seed
```
5. Поднять локально веб-сервер
```
php artisan serve
```
6. Готово! доступно на http://127.0.0.1:8000
